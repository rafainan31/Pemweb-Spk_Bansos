# Sistem Kelayakan Bansos TOPSIS

Project ini berisi website SPK kelayakan bansos dengan:

- Frontend: React + TypeScript + Vite
- Backend: Express + TypeScript
- Database: MariaDB
- Metode SPK:SAW, WP, TOPSIS
